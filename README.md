Descripcion general
Una aplicación web simple y responsiva diseñada para el seguimiento
de hábitos saludables y monitoreo del estado de ánimo diario.
Construida pensando en la intersección entre tecnología y bienestar
personal.

Tecnologias que se utilizaron

React (Hooks): Gestión de estado con useState y efectos secundarios con useEffect.
Tailwind CSS: Para un diseño de interfaz de usuario (UI) limpio, responsivo y moderno.
JavaScript (ES6+): Lógica de manipulación de arrays y eventos.
Lucide React: Iconografía ligera y escalable.

Uso

Gestión de Tareas: Añadir, completar y eliminar metas diarias de bienestar.
Seguimiento del Estado de Ánimo: Selector interactivo de emociones.
Progreso Visual: Barra de progreso dinámica basada en tareas completadas.
Diseño Responsivo: Interfaz optimizada para móvil y escritorio.
